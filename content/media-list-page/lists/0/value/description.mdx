Die Medienliste für amc\_4.3 enthält folgende Neuerungen: Schon seit dem Umstieg auf amc\_4.2 ist nach 2020 der Medientyp `tv` nur noch durch Daten aus dem ORF vertreten (`MWVOLL`), die zudem noch durch automatische Transkriptionsverfahren bereitgestellt wurden.

In den frischen Daten ab amc\_4.3 — also Daten aus 2023 und jünger — werden die ORF-Daten genauer differenziert: das frühere einheitliche `MWVOLL` für alle ORF Daten wird nun unterteilt in die einzelnen Sendeanstalten: vgl. `BEEYMOFOE1`, `BEEYMOFOE3`, `BEEYMOFORF1` etc. (Der einheitliche Namensbestandteil BEEYMOF wurde von der APA vergeben und macht explizit, dass die Transkriptionen automatisch mit einem Tool der Firma [Beey](https://www.beey.io/) erzeugt wurden)

Eine weitere Konsequenz dieser Differenzierung ist, dass nun für die jetzt klar erkenntlichen Radiosender ein neuer Medientyp radio eingeführt wurde.
